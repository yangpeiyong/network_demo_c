# 1. One to One TCP

You can read the source code `tcp_server.c` and `tcp_client.c`

`tcp_server.c` for tcp_server.

`tcp_client.c` for tcp_client.

*Build*

clang -std=c11 -g -o tcp_server tcp_server.c

clang -std=c11 -g -o tcp_client tcp_client.c

*RUN*

run server first:

`./tcp_server`

then run client:

`./tcp_client`
